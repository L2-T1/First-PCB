# First-PCB
This my first PCB for learning and for the Solder activity ran by Hack Club

# Description
This project is creating a circut board that powers 2 LED's with one battery and two resistors 

# Bill of materials
- 2x LED's
- 2x Resisters
- 1x Single Battery-Cell
